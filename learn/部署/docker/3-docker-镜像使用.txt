1.列出本地镜像：
docker images
repository：镜像仓库源
tag：镜像标签
image id：镜像id
created：镜像创建时间
size：镜像大小
2.从仓库下载镜像
docker pull repository:tag
如果不指定tag将使用latest版本
3.查找镜像
docker search httpd
4.删除镜像
docker rmi hello-world
5.创建镜像
--1.更新镜像
更新镜像前创建一个容器：docker run -t -i ubuntu:15.10 /bin/bash
在容器内使用apt-get update命令进行更新
完成操作后exit推出容器
通过docker commit提交容器副本：docker commit -m="has update" -a="runoob" e218edb10161 runoob/ubuntu:v2
-m：描述信息
-a：指定镜像作者
e218edb10161：容器ID
runoob/ubuntu:v2 ：要创建的镜像名

--2.构建镜像
使用docker build，从零开始创建一个新的镜像。为此要创建一个Dockerfile文件。文件中包含一组指令告诉docker如何构建我们的镜像。
编写好dockerFile后使用docker build
docker build -t runoob/centos6.7 .
注意最后的. 这个. 就是指dockerfile的位置，可以使用绝对路径，如果现在就在dockerfile的路径下可以用.代替

--3.设置镜像标签
docker tag 860c279d2fec mysql:dev
860c279d2fec 是镜像id
mysql是镜像名
dev就是要添加的tag
